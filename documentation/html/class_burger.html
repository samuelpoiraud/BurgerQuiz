Detailed Description

Cette classe encapsule les données relatives à un burger, son état courant, l'état de sa LED et son équipe. C'est un proxy qui masque les données encapsulées par la classe BurgerData.

See also
BurgerData. L'instance de BurgerData est stockée sous la forme d'un pointeur partagé sans copie à l'écriture. Ainsi les modifications sur l'état du burger sont reportées partout dans l'application.
